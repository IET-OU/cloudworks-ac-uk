Cloudscape created by:

Gráinne Conole
11 October 2009

This cloudscape has been set up to collate and discus pedagogical pattern tools resource and methods. Feel free to add resources, activities, tools and methods for design that you are aware of, and please name and tag them so they can be found easily.

  • Activities = ideas, designs, schedules of activities which can be used to promote thinking and understanding around pedagogical patterns.
  • Methods = different  methodologies
  • Resources = handouts, diagrams, papers, flash cards etc
  • Tools = software tools which use or support pedagogical patterns concepts

Feel free to add resources, activities, tools and methods that you are aware of, and please name and tag them using the key below so they can be found easily. Contributing 3 objects to this resource bank makes you eligable for the OLDS-MOOC badge 'Resource Gather' (click on the badge to apply).
