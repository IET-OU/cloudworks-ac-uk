This cloudscape has been set up for presentations and questions associated with the Massey University Blended Learning conference. 

Purpose of the forum

‘To exchange perspectives, establish a common understanding and identify our shared development needs for a blended learning institution’.

Approach: collective voices, shared concerns, common solutions.
