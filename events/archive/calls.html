How to add an event

To add a conference to this list, create a cloudscape for the event and add the dates and location when you create the cloudscape.

To add a workshop, seminar or talk to this list, create a cloud, then edit the cloud to add the event date.

To add a deadline to this list, create a cloud, then edit the cloud to add the deadline.
