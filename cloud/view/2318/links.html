All your OER repositories in one place

Cloud created by:

This cloud aims to aggregate links to useful OER repositories. Please add any that you come across - where appropriate please detail the subject area and age/ level targeted by the repository.

Extra content

There is an overlapping area of open access repositories which contains scholarly output. In amongst these resources there are resources which have been reversioned into teaching resources. And of course the scholarly resources themselves can be used for teaching. OPEN DOAR (http://www.opendoar.org/index.html), the directory of open access resources, provides "learning objects" as a possible search choice.
