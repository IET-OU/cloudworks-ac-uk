DANI'S NOTES FROM SESSION: ADVANCED NROC

 

Integration of NROCmaterials into school and state curriculum – Bob Cole (MD)

  • Scaffolding includes more formative assessment, more overviews
  • Summative assessment from NROC left intact along with overview, and media lessons
  • Did not always want to use all components of mm lesson (too much)
  • Have more control over presentation once formatted (scaffolding completed) for specific LMS
  • Separated out “slides” within a lesson in Biology – added objectives to single slide as well as formative assessment from state assessment/question bank
  • Added closed captioning to some lessons; pulled out extra info (particularly from AP courses) that would not be assessed in traditional course

 

Challenges:

  • Flash can be more costly (need designer)
  • Developing different “styles” of design for course additions and/or customizations (is template/player the solution?)
  • Smaller learning objects for different student audiences
  •  Repository and metadata at more granular levelto support easy searching and embedding of content (subject searching)
  • OPI standard for federated searching for repository (when searching at state DOE site – also automatically searching NROC)

 

Question from group:  can NROC break content into smallerchunks and make them “linkable” from our repository rather than these materialshaving to be hosted at member institutions?

 

NROC Player/Editordemo and discussion – Renae Bent and Brian Rowlett

  • New “skins” will be available soon for backgrounds and titles
  • Player/editor will be upgraded as Algebra project goes forward
  • Members should submit a request to get the player/editor to Brian Rowlett
  • Request to be able to insert podcast material into player/editor

 

Metadata, what iscurrently available – what do you need? – Brian Rowlett

  • Most metadata has come from mining content received after the fact; hand-tagging
  • IEEE LOM was starting point and pulled info we thought would be most useful – tagged at the course-level
  • Mapping available in Excel on NROC network – if anyone needs xml or other format we can create that – requests should go to Brian
  • Metadata currently available at lowest level – keywords, titling info, data about where key info is within each course
  • Correlations to state standards in our database but don’t have exportable reports written yet.  What type of reports do members need?
  • We are interested in making web (self-serve) reporting available to members if they believe this would be a useful tool
  • Projects taken on “as-needed” for and with member partners – federated searching
