Dianne Johnson Review of E Design Template

Readability, expressiveness and utility

This is a n interesting representation to use as it is very similar to a Scheme of Work template that we use in college. I found the table format easy to read and understandable. It shows that there are gaps in the consolidation of learning that could be dealt with by the tutor. I like the way that it uses a spiral learning process starting with tutor led closed activities up to student led open activities. My learning design did incorporate parts of facilitated investigation but this needs careful planning or you will not succeed as my exam results show. I think this model would be very useful for a research project type activity but not all activities are relevant to this – maybe this model could span a number of lessons to show the progress.

Limitations

This design would have a more limited use for less able and lower level learners as independent work happens less at these levels. Although, I do remember being given projects at school at an early age!! It would be more useful in a face to face scenario as there is less opportunity for collaboration.

 

Review of 4SPPICes Model

Readability, expressiveness and utility

I love the visual representation on this model and find it is easy to read. I particularly like its simplicity and the fact that you can easily see the methods listed. The collaboration tool is excellent. A really useful extra dimension is the ‘history’ section which forces the practitioner to think about where any outcomes and materials from the learning will be accessed and stored. This is an important point that I did think about for my own learning design.

Limitations               

There is no reflection section nor is there any link to aims, objectives of the session or success criteria. I would need to use additional paperwork to ensure that each method delivers the learning that I want it to.
