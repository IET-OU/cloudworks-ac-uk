From website:

 

The Visual Understanding Environment (VUE) is an Open Source

project based at Tufts University. The VUE project is focused on

reating flexible tools for managing and integrating digital resources

in support of teaching, learning and research. VUE provides a flexible visual environment for structuring, presenting, and sharing digital information.

I use VUE based concept mapping to represent the curriculum when designing visual gateways to online learning resources.

The JISC eTutor project is using a discovery learning pedagogic approach for online distance learners. A competence map has been created using the VUE application with nodes linked to customised and user-refinable search engines designed to home in on relevant learning resources.

See: http://e-college.wetpaint.com/page/Word+Processing+Competence+Map
