An overview of the OU Learning Design Iniative

Cloud created by:

Gráinne Conole
21 November 2008

The Open University Learning Design Initiative started in April 2007 with strategic funding. The project aim is to develop and implement a methodology for learning design composed of tools, practice and other innovation that both builds upon, and contributes to, existing academic and practitioner research.

Facets of OULDI work:

  • Evidence base
  • Connection
  • Representation
  • Collaboration

Links are provided to a number of key presentations or sites of relevance.
