EOR: Example uses of the Ecology of Resources Design Framework

  • Favourite
  • 1201 views
  • 2 favourites

Cloud created by:

Joshua Underwood
22 November 2012

Skim through the presentations (embedded below) to get a feel for how the EoR design framework has been used in:

  • design of support for better connecting primary school and home numeracy learning using tablet PCs.
  • co-design of technology-enhanced learning about sustainability with teenagers.
  • co-design of BYOD support for self-managed learning field trips with teenagers.
  • workshops with teachers designing technology-enhanced participatory science inquiry activities for secondary & primary school.
  • participatory design of an Android app for foreign language vocabulary learning with self-directed adult learners.
For a more detailed description of the framework and worked examples see the EoR Design Framework & Models Wiki.

These examples are used with EoR Phase 1 Cloudscape
