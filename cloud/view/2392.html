Irish Learning Technology Association (ILTA) web site launch, 1/10/09

Cloud created by:

Gráinne Conole
1 October 2009

This cloud has been set up for the launch of the Irish Learning Technology Association (ILTA) web site launch on 1st October 2009. It includes the slides from Gráinne Conole's introductory presentation and a set of questions for discussion:

  • How can we foster and support the development of the Irish Learning Technology community, how can we encourage better sharing and discussion?
  • How can web 2.0 technologies help and in particular how might cloudworks help?
