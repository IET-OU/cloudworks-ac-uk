Video Interview Recording

  • Favourite
  • 714 views

Cloud created by:

Will Pollard
26 September 2010

I notice there are YouTube interviews starting to appear on Cloudworks. i am trying to do some myself so start this cloud hoping for guidance on how to do it better. I think it needs more than one person, or concentration on the camera rather than the questions. (They could be edited in later) I usually don't use a tripod so there is some shake but the movement helps momentum. Sound is a problem. I try to get the mike on the camera roughly the same distance between me and the person being interviewed. External mikes could help but more complex to set up. Links could include examples as well as suggestions.
