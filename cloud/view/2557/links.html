Resources from Brunel University

Cloud created by:

This Cloud has been set up to aggregate tools, resources and discussion around using the library and media service teams at Brunel University to support the design and implementation of new or revisited modules. In addition, this stall will begin to highlight the information and tools available for addressing accessability issues early in the design process.
