(Wednesday, November 16, 2011) Hands-on session with Sofocles tools

  • Favourite
  • 813 views

Cloud created by:

Luis P. Prieto
11 November 2011

In this session, we will try some of the technological tools developed by the Sofocles project, for the deployment of learning designs and integration of external tools in VLEs (e.g. Moodle). To do that, we will try to implement a real-world CSCL scenario for higher education, using the tools.
