Design-Practice Activity 3 - 'Representing Designs'

  • Favourite
  • 1229 views

Cloud created by:

Paul Mundin
5 April 2011

This activity will take about an hour.

  • Read the Resource (Conole, 2009) 'An overview of design representations'

This document builds on the Resource 'The Design Lifecycle' and looks at the ways a learning design initiated during the 'Visioning' stage and then iteratively developed during the 'Gathering' and 'Assembling' stages could be represented, either textually or visually.

  • Do you create and/or use any learning activity representations textually or visually?
  • Would you find find designing a learning activity using a visual representation helpful?
  • What are the advantages or disadvantages of creating and using visual representations?
  • What tool(s) do you use to create a design representation?

You could:

  • Post/share your thoughts about the above questions or the document in this Cloud
  • If you have one - embed one of your own visual representations with a brief description - for sharing with the other participants
  • Spend some time reading and responding to the postings of other participants
  • Respond to postings about your comments
