This is an ice-breaking activity for participants to get to know each other better

Cloud created by:

Andreia Inamorato Dos Santos
3 December 2009

How about finding out a bit more about the people you work with on a daily basis (and also about the ones you are just coming to know through this event)?

Please state three interesting things about you, one of which is a lie. 

We'll all try to identify which one is the lie, let's see how well we can spot it!

 Let's have some fun...
