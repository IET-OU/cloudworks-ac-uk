Embedding digital and information literacy into outcomes, assessment and learning for OU qualifications

http://www.open.ac.uk/libraryservices/pages/DILFramework

Cloud created by:

Katharine Reedy
12 January 2013

This is my Dreambazaar project proposal. It's about coming up with a design to embed digital and information literacy into degree programmes so that these skills and capabilities are fully integrated into learning, teaching and assessment.

If anyone else is interested in that, I'd like to be in your gang!

I've added a link to the OU's Digital and Information Literacy Framework in case you'd like to see what we've come up with so far. We think it could be a useful tool for learning design.

More later......
