Características de la Web 2.0 y la revisión de herramientas a partir de una clasificación previa de las mismas

Cloud created by:

Maria Eugenia Morales Mora
13 November 2009

La Web 1.0 era estática, no permitía la interacción y era muy limitada en cuanto a las posibilidades del usuario para publicar contenidos, participar en comunidades con otros y acceder a la información.

La web 2.0 o web colaborativa, permite a todos los usuarios publicar contenidos de diversa índole, participar en comunidades virtuales y trabajar en forma colaborativa... dispone de múltiples herramientas de gran útilidad en el campo educativo.

La web. 30 o web semántica, se perfila como la web del conocimiento, en el cual todos construimos conocimiento en forma colaborativa y accedemos a la información organizada con fácil acceso y uso intuitivo... responde a patrones de búsqueda de información y permite tejer redes de conocimiento.
