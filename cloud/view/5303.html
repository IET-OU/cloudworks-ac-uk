Cloud created by:

Paul Mundin
8 April 2011

Thank-you for completing Design-Practice Activity 4. We would like you to spend a few minutes commenting on your experience of this activity - in this Cloud - 'Contribute to the discussion'. In particular we are interested in:

  1. How long did you take to complete the Activity?
  2. Were the instructions for the Activity clear? If they weren't how could they be improved?
  3. Was the content of the Activity informative? If not what would you have liked to see?
  4. What did you like about the Activity?
  5. What did you dislike about the Activity?
  6. Did you post any comments to the Activity Cloud? If you did not post any comments can you say why you didn't?
  7. If you did post comments to the Activity Cloud did you get any feedback?
  8. How useful was the feedback?
  9. Do you have any other comments you would like to make?

Thank-you for taking the time to respond. This feedback will help us improve the module.

How long did you take to complete the Activity? probably about two and a half hours

Were the instructions for the Activity clear? yes; however, I was not completely sure about the intention was of the slides of what appeared to be road maps

Was the content of the Activity informative? yes; it might have been useful to have an example of a Module Map that represents an existing OU module

What did you like about the Activity? it provided me with an opportunity to identify the specific components of the OU module on which I tutor using the six elements included in the Module Map exemplar:  Guidance and Support; Content and Activities; Communication and Collaboration; Reflection and Demonstration; Module Summary; Key Words; I suggest that this process could be an effective tool for me as an AL in:  highlighting the structure of the module and relationships between various elements of the module design; support reflection on the learning design of the module; sharing the overall design of the module in a visual way with students; providing opportunities for student input; organising my  thoughts; capturing a tangible record of my understanding of and response to the overall design of the module at a particular moment in time

What did you dislike about the Activity?  the awkwardness of including graphics using the comment facility; the preset line spacing in the comment facility

Did you post any comments to the Activity Cloud?  yes

If you did post comments to the Activity Cloud did you get any feedback?  not yet

How useful was the feedback?  not  applicable

Do you have any other comments you would like to make?  I look forward to reading additional comments on this discussion site and participating in the face-to-face workshop in Milton Keynes, 10 June

I note here that because of other work commitments that now require my attention; I will now not be in a position to spend much time with this project.  I will dip into the project in order to view the progress from time to time within the next week so that I am aware of subsequent additions and information relating to the face-to-face workshop on the 10th of June.

  1. How long did you take to complete the Activity? maybe up to 3 hours, as I did it in chunks on different days
  2. Were the instructions for the Activity clear? Yes, I think so, once I understood it didn't have to be a whole course. 
  3. Was the content of the Activity informative? Yes, I wouldn't want more, as there was a good deal of info. to take in already.
  4. What did you like about the Activity? I liked it better once I'd actually had a go, and realised how it helped me think in more depth about the whole process of planning for a talk I'm about to give.
  5. What did you dislike about the Activity? Having to get it done in time, in a context when I am already quite pressed.
  6. Did you post any comments to the Activity Cloud? Yes
  7. If you did post comments to the Activity Cloud did you get any feedback? I did, and these were very helpful.
  8. How useful was the feedback? I had expressed a feeling of dauntedness when i saw how much material the first comment included, and the writer returned to reassure me, which is a nice example of 'social presence'; similarly, as I also mentioned that the slide kept being interfered with as business videos kept popping up, another comment arrived from another participant to explain why this was happening.
  9. Do you have any other comments you would like to make? The sense of exposure, not only to one's colleagues in this learning design training, but also the thought that anyone can view what we write: that is quite a feature, that affects what I write.
